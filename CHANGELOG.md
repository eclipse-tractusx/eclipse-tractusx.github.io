# Changelog

All notable changes to the Eclipse Tractus-X Webpage will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [R25.12] - 2025-11-20

### Added

- Added new KIT home page with the following features:
  - Architecture view with reusable components for dataspaces and use cases (Dataspace Foundation, Industry Core Foundation, Cross-Datasapce Foundation, Industry Specific Use Cases) with filtered pages
  - Transformed KIT Images into flexible CSS objects
  - Added SVG KIT logos in standalone format.
  - Added KIT footer
  - Added KIT Dataspace Filter Pages
  - Added industry specific filtered pages
  - Added flexible kit description cards
- Added KIT master data
  - Added schema and validator on website build and development time.
- Improved KIT Navbar
  - Added Script to autogenerate the KIT navbar based on the KIT master data on building time
- Added contribution guideline (with flexible items)
- Added TRG 10.XX for the KITs, based on TRG 9-01 (which was in draft)
  - TRG 10.01 - KIT Architecture (Active)
  - TRG 10.02 - KIT Content (Active)
  - TRG 10.03 - KIT Lifecycle (Active)
  - TRG 10.04 - Graduation Process (Draft) - Needs feedback from the community
  - TRG 10.05 - Deprecation Process (Draft) - Needs feedback from the community.
- Added Changelog
- Improved the documentation adding new views:
  -  KIT master data overview
  -  KIT Statistics and Analitics
  -  KIT Lifecycle
  -  KIT Framework
  -  KIT Getting Started
- Improved KIT content structure to be multi-industry and multi-dataspace
- Improved KIT overall architecture to be more modular.

### Updated

- Refactored KIT architecture module to be multi-industry and multi-dataspace.
- Increased the documentation available for the KITs, with explanations, content guidelines, diagrams and examples.
- Refactored main landing page to include the KITs 2.0 structure.
- Improved Eclipse Tractus-X overall explanations.
  
### Fixed

- Fixed broken links
- Remove many Catena-X specific references from the consortia time.
