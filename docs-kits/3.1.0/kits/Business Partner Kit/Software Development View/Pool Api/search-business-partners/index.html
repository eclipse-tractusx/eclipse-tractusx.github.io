Version: 3.1.0

Get page of business partners matching the search criteria​

deprecated

This endpoint has been deprecated and may be removed in future versions of the API.

This endpoint tries to find matches among all existing business partners, filtering out partners which entirely do not match and ranking the remaining partners according to the accuracy of the match. The match of a partner is better the higher its relevancy score.

Query Parameters
  • name string

    Filter legal entities by name

  • legalForm string

    Filter legal entities by legal form name

  • status string

    Filter legal entities by status official denotation

  • classification string

    Filter legal entities by classification denotation

  • administrativeArea string

    Filter business partners by administrative area name

  • postCode string

    Filter business partners by postcode or postcodes

  • locality string

    Filter business partners by locality full denotation

  • thoroughfare string

    Filter business partners by thoroughfare full denotation

  • premise string

    Filter business partners by premise full denotation

  • postalDeliveryPoint string

    Filter business partners by postal delivery point full denotation

  • siteName string

    Filter sites by name

  • page string

    Default value: 0

    Number of page to get results from

  • size string

    Possible values: <= 100

    Default value: 10

    Size of each page

Responses
  • 200
  • 400

Page of business partners matching the search criteria, may be empty
