Find best matches for given text in business partner legal forms​

Performs search on legal form names in order to find the best matches for the given text. By specifying further request parameters the set of business partners to search in can be restricted. If no text is given, the endpoint lists possible legal form names in the search set.

Query Parameters
  • text string

    Show legal form names best matching this text

  • name string

    Filter legal entities by name

  • legalForm string

    Filter legal entities by legal form name

  • status string

    Filter legal entities by status official denotation

  • classification string

    Filter legal entities by classification denotation

  • administrativeArea string

    Filter business partners by administrative area name

  • postCode string

    Filter business partners by postcode or postcodes

  • locality string

    Filter business partners by locality full denotation

  • thoroughfare string

    Filter business partners by thoroughfare full denotation

  • premise string

    Filter business partners by premise full denotation

  • postalDeliveryPoint string

    Filter business partners by postal delivery point full denotation

  • siteName string

    Filter sites by name

  • page string

    Default value: 0

    Number of page to get results from

  • size string

    Possible values: <= 100

    Default value: 10

    Size of each page

Responses
  • 200

Best matches found, may be empty
