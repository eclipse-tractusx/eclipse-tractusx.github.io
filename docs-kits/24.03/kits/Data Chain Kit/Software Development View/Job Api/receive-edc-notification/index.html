Accepts notifications sent via EDC.​

Accepts notifications via EDC. Notifications are filtered by their type and processed accordingly.

Request Body required
  • content object required
  • hops int32
  • bpn string
  • result string
  • header object required

    EDC notification header information

  • notificationId string required
  • notificationType string required
  • originalNotificationId string
  • recipientBpn string required
  • replyAssetId string
  • replyAssetSubPath string
  • senderBpn string required
  • senderEdc string
