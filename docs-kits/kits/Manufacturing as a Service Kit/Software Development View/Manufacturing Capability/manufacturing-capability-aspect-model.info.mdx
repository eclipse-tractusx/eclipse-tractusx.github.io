# Manufacturing Capability Aspect Model



The Manufacturing Capability API description below shows how the Manufacturing Capability Aspect Model could be used. The aspect model is designed to represent manufacturing capabilities based on the concepts of products, processes, resources and capabilities and their relations to each other. Depending on the use case different request paths can be implemented, based on the specific MaaS implementations.
