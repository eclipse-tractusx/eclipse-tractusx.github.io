get_pcf​

get_pcf

Path Parameters
  • productId string required

    ID of the product/material the PCF is requested for

    Example: SomeProductID
Query Parameters
  • BPN string required

    The caller's Catena-X BusinessPartnerNumber

    Example: BPN9354218404L
  • requestId string required

    ID identifying the call (will be referenced in corresponding PCF response)

    Example: X374e9453K
  • message string

    URL encoded, max 250 chars

    Example: No%20offset%20included%2C%20please%21
Responses
  • 202

PCF was accepted. PCF will be sent later via to POST endpoint.
