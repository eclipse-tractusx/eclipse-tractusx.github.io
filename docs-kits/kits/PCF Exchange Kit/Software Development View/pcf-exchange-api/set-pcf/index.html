set_pcf​

set_pcf

Path Parameters
  • productId string required

    ID of the product/material the PCF referring to

    Example: SomeProductID
Query Parameters
  • BPN string required

    The caller's Catena-X BusinessPartnerNumber

    Example: BPN83456784L
  • requestId string

    ID identifying the request call (same as within original PCF request), if the PUT is responding to a call. Can be dismissed in a PCF update call.

    Example: X374e9453K
  • application/json
Request Body required

The requested PCF in WBCSD format
