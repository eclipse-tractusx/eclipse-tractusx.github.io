  • totalElements int64

    Total number of all results in all pages

  • totalPages int32

    Total number pages

  • page int32

    Current page number

  • contentSize int32

    Number of results in the page

  • content object[]

    Collection of results in the page

  • technicalKey string

    The technical identifier (unique in combination with businessPartnerType).

  • businessPartnerType string

    Possible values: [LEGAL_ENTITY, ADDRESS]

    Specifies if an identifier type is valid for legal entities (L) or addresses (A). Sites (S) are not supported.
