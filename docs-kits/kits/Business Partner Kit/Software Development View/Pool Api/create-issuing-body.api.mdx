## Create new issuing body



Create a new issuing body which can be referenced by business partner records. An issuing body should be an entity which the Catena organisation trusts to issue identifiers.The actual name of the issuing body is free to choose and doesn't need to be unique. The technical key can be freely chosen but needs to be unique as it is used as reference by the business partner records. A recommendation for technical keys: They should be short, descriptive and use a restricted common character set in order to ensure compatibility with older systems.
