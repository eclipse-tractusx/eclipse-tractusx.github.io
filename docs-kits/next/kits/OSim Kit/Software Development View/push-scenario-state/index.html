pushScenarioState​

The pushScenarioState enables distribution of a new scenario status and thus a notification of the procedure with the planned scenario

Path Parameters
  • bpns string required
    Example: BPNS00000007OTZ3
  • scenarioId string required
    Example: 50737df3-4237-4652-b092-1ef8649f6ca6
  • state string required

    Possible values: [InRealization, Processed, Canceled]

    Example: InRealization
