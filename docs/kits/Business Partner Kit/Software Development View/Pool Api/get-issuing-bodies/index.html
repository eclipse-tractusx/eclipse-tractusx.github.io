Get page of issuing bodies​

Lists all currently known issuing bodies in a paginated result

Query Parameters
  • page string

    Default value: 0

    Number of page to get results from

  • size string

    Possible values: <= 100

    Default value: 10

    Size of each page

Responses
  • 200
  • 400

Page of existing issuing bodies, may be empty
