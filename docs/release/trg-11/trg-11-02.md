## Why

Eclipse Tractus-X is an open source project hosted by the Eclipse Foundation licensed under the Apache License 2.0 ([Apache-2.0](https://spdx.org/licenses/Apache-2.0)). For non-code the default license is the Creative Commons Attribution 4.0 International ([CC-BY-4.0](https://spdx.org/licenses/CC-BY-4.0.html)).

When **substantial amounts** of content are generated with the assistance of Generative AI, the Eclipse Foundation FOSS governance requires that the AI-generated portions are disclosed and made available under the public-domain dedication [CC0-1.0](https://spdx.org/licenses/CC0-1.0.html), so that they are not subject to copyright claims that could conflict with the project's license.

The source of truth is always the [Eclipse Foundation Project Handbook — Generative AI](https://www.eclipse.org/projects/handbook/#genai). The usage rules for Generative AI are described in [TRG 11.01 - Generative AI usage and contribution](/docs/release/trg-11/trg-11-01).

:::info

The requirements described here **must** be met for each contribution that contains substantial AI-generated content.

:::

## Description

When a file contains substantial amounts of content generated with the assistance of Generative AI, the file's header **must** be extended with:

- An **AI Disclosure** statement describing how much of the file was AI-generated, which tool generated it, and that the AI-generated portions are made available under `CC0-1.0`.
- A combined **SPDX-License-Identifier** that lists the project's license **and** `CC0-1.0` (for example `Apache-2.0 AND CC0-1.0`).
- An **`Assisted-by:`** line naming the tool and the concrete model (including its version) that was used — mirroring the commit trailer described in [TRG 11.01](/docs/release/trg-11/trg-11-01).

**Notes:**

- The disclosure describes the **degree** of AI generation. Choose one of `minimally`, `largely`, or `entirely`.
- The **year** is the year of the initial creation, following [TRG 7.02](/docs/release/trg-7/trg-7-02).
- Never change the rest of the license section, except for the comment characters appropriate to the language.

## AI disclosure header

### Example for Apache-2.0 code (Java)

```md
/********************************************************************************
 * Copyright (c) 2026 Contributors to the Eclipse Foundation
 *
 * See the NOTICE file(s) distributed with this work for additional
 * information regarding copyright ownership.
 *
 * This program and the accompanying materials are made available under the
 * terms of the Apache License, Version 2.0 which is available at
 * https://www.apache.org/licenses/LICENSE-2.0.
 *
 * Unless required by applicable law or agreed to in writing, software
 * distributed under the License is distributed on an "AS IS" BASIS, WITHOUT
 * WARRANTIES OR CONDITIONS OF ANY KIND, either express or implied. See the
 * License for the specific language governing permissions and limitations
 * under the License.
 *
 * AI Disclosure: This file was [minimally | largely | entirely] AI-generated.
 * The AI-generated portions are made available under CC0-1.0 and not subject
 * to the project's license. The human contributor has reviewed and verified
 * that the code is correct.
 *
 * SPDX-License-Identifier: Apache-2.0 AND CC0-1.0
 * Assisted-by: GitHub Copilot (Claude-Opus-4.8-2026-06-16)
 ********************************************************************************/
 ```

### Example for non-code (CC-BY-4.0)

```md
#######################################################################
# Copyright (c) 2026 Contributors to the Eclipse Foundation
#
# See the NOTICE file(s) distributed with this work for additional
# information regarding copyright ownership.
#
# This work is made available under the terms of the
# Creative Commons Attribution 4.0 International (CC-BY-4.0) license,
# which is available at
# https://creativecommons.org/licenses/by/4.0/legalcode.
#
# AI Disclosure: This file was [minimally | largely | entirely] AI-generated.
# The AI-generated portions are made available under CC0-1.0 and not subject
# to the project's license. The human contributor has reviewed and verified
# that the content is correct.
#
# SPDX-License-Identifier: CC-BY-4.0 AND CC0-1.0
# Assisted-by: GitHub Copilot (Claude-Opus-4.8-2026-06-16)
#######################################################################
 ```

:::tip

Use your IDE's template mechanism to add the disclosure text automatically when you create AI-assisted files, and remember to also add the matching `Assisted-by:` trailer to your commit as described in [TRG 11.01](/docs/release/trg-11/trg-11-01).

:::
